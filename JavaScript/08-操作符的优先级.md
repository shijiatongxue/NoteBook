# 操作符的优先级

| 操作符                  | 作用               |
| ----------------------- | ------------------ |
| . [] ()                 | 提取属性与调用函数 |
| delete new typeof + - ! | 一元运算符         |
| * / %                   | 乘法、除法、求余   |
| +-                      | 加法/连接、减法    |
| >= <= > <               | 不等式运算符       |
| === !==                 | 等式运算符         |
| &&                      | 逻辑与             |
| \|\|                    | 逻辑或             |
| ?:                      | 三元               |
