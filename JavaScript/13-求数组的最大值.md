# Math.max.apply
```js
var arr = [1,2,3,4,2,1,-1];
var max = Math.max.apply(null, arr);
console.log(max);
// => 4
```
# 比较法
```js
var arr = [1,2,3,4,2,1,-1];
Array.prototype.max = function(){
    var max = -Infinity;
    for(var i = 0, len = this.length; i < len; i++){
        if (this[i] > max){
            max = this[i];
        }
    }
    return max;
}
```
