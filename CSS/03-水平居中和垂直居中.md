# 水平居中和垂直居中

## 水平居中

- 行内元素水平居中

```css
/* 给父元素设置text-align: center */

.container{
  text-align: center;
}
```

- 块级元素水平居中

```
/* 定宽块 */
.container{
	width: 100px;
	margin: 0 atuo;
}

/* 不定宽块 */
.container{
	text-align: center;
}
.item{
	display: inline-block;
}

/* 或 */
.container{
	display: flex;
	justify-content: center;
}
```

## 垂直居中

- 单行文本

```css
.item{
  height: 24px;
  line-height: 24px;
}
```

- 多行文本

```css
.container{
  display: table;
}
.item{
  vertical-align: middle;
  /* 或 */
  display: table-cell;
}
```

- 块级元素

```css
.container{
  display: flex;
  align-items: center;
}
```
