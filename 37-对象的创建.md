# 字面量
通过字面量创建的对象，它们的原型是Object.prototype。
```js
var obj = {name: 'shijia'};

// obj.__proto__ === Object.prototype;
// true
```

# new
它们的原型属性指向构造函数的原型。
```js
function Person(){
    name = 'shi';
}

var student = new Person();

// student.__proto__ === Person.prototype;
// true
```

# Object.create(proto)
它参入的参数就是创建出来对象的__proto__。
```js
function Person(){
    name = 'shi';
}

var student = Object.create(Person.prototype);

// student.__proto__ === Person.prototype;
// true
```
