# 科里化

只传递一部分参数，让它返回一个函数去处理剩下的参数。

```js
// 求和

function add() {
  let sum = 0;
  sum = arguments.length > 0 ? [...arguments].reduce((a, b) => { return a+b}, sum) : sum;
  
  return function fc() {
  	if (arguments.length === 0) {
   		return sum;
    } else {
   		sum = [...arguments].reduce((a, b) => { return a+b}, sum);
      return fc;
    }
  }
}
```
